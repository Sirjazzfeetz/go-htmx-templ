# Go + HTMX + Templ Starter

Minimal HOWL-compliant web stack using:
- Go (net/http)
- HTMX (CDN)
- Templ (type-safe templates)
- Tailwind (CDN)

## Getting Started

```bash
go install github.com/a-h/templ/cmd/templ@latest
templ generate
go run ./cmd/server
```

Then visit [http://localhost:3000](http://localhost:3000)
